# Benchmark / Demo

这个目录提供一套小型中文 synthetic benchmark/demo，用于说明行为和回归验证，不包含真实私有论文内容。

## 文件

- `cases.md`：benchmark case 定义
- `results.md`：期望行为和样例 before/after

## 覆盖内容

- 中文摘要去套话
- 中文引言去模板化
- 中文相关工作归因安全压缩
- 中文方法高风险微调
- 中文结果与结论的论断档位控制
- 中文表层卫生清理
- 已经自然文本的不改能力
- 不匹配请求的拒绝或降级

它是一个开源说明性产物，不是完整的中文学术写作 benchmark。
