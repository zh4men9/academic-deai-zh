# Benchmark / Demo

这个目录提供一套小型 synthetic benchmark/demo，用于说明行为和回归验证，不包含私有论文内容。

## 文件

- `cases.md`：benchmark case 定义
- `results.md`：期望行为和样例 before/after

## 覆盖内容

- 低风险去模板化
- detail-preserving compression
- citation-sensitive compression
- symbol-safe high-risk micro-edits
- surface hygiene cleanup
- claim calibration 对比
- already-natural prose 的 no-op
- mismatch handling

它是一个开源说明性产物，不是完整的 academic writing benchmark。
